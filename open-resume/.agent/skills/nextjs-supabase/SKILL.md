# Next.js + Supabase Skill

## When to Use
When implementing database operations, authentication, or Supabase integration in Next.js.

## Guidelines
- Use Server Components for data fetching where possible
- Use Route Handlers for API endpoints
- Implement proper error boundaries
- Follow TypeScript strict typing

## Scripts
- `scripts/generate-types.ts` - Generate Supabase types
- `scripts/seed-db.ts` - Seed database for testing
